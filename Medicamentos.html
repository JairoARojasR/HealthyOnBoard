Medicamentos

Azacitidina

Funciona al desacelerar o detener el crecimiento de las células cancerosas en su cuerpo.

Dacarbazina

Se usa para tratar el melanoma (cáncer de piel) que propago a otras partes del cuerpo.

Temozolomida

La Tezolomida se utiliza para tratar ciertos tipos de tumores cerebrales.

Carmustina

Se usa para el tratamiento de ciertos tipos de tumores de encéfalo.

Lomustina

Se usa con otros medicamentos para tratar el linfoma de Hodgkin que no haya mejorado.

Estreptozocina

se utiliza para tratar el cáncer de páncreas que se ha propagado a otras partes del cuerpo.

Azacitidina

Se usa para el tratamiento del síndrome mielodisplásico (globulos rojos anormales).

Fludarabina

Se usa para tratar la leucemia linfocítica crónica (cancer de globulos blancos)

Metotrexato

El metotrexato trata el cáncer retardando el crecimiento de células cancerosas.

Epirubicina

Se usa en combinación con otros medicamentos para tratar el cáncer de seno.

Valrubicina

Se usa para tratar un tipo de cáncer de vejiga urinaria no tratada.

Mitomicina C

Es un tipo de antibiótico que solo se usa en la quimioterapia contra el cáncer
